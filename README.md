# Kuz-Ram-Fragment-Size-Predictor
⛏️ Kuz-Ram Fragment Size Predictor built with Python and Streamlit. This app predicts mean fragment size using the Kuz-Ram model, generates Rosin-Rammler distribution curves, calculates P20, P50, and P80 values, and classifies blast quality. A MiningTech project combining mining engineering and data-driven analytics.
